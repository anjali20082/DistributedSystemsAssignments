
1. First the MultithreadedServer_MT20082.java file is to be run. Then it waits for clients to connect.
2. Then Client_MT20082.java file is to be run. The client sends a text message to server.
3. The Client_MT20082.java file can be run again as a different client that wants to connect to the server.
4. The server will print each client's message appenede with client's name on the console and also server sends the same    string appended by client's name that was sent by a particular client as a reply to that client.
